# Online Retail Sales Analysis

## 📌 Project Overview

This project explores an online retail transactions dataset to uncover customer purchasing behavior, sales trends, and business performance through Exploratory Data Analysis (EDA).

The goal is to transform raw transactional data into actionable business insights that support strategic decision-making.

---

## 🎯 Objectives

- Explore retail transaction data
- Clean and prepare the dataset
- Analyze customer purchasing patterns
- Identify top-selling products
- Measure sales performance
- Discover revenue trends

---

## 📂 Dataset

The dataset contains online retail transactions including:

- Invoice Information
- Products
- Quantity Sold
- Unit Price
- Customer IDs
- Purchase Date
- Country
